== Installation ==

**Must have a widget enabled theme for this plugin to work**

1. Upload `gravy-events` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Navigate to Appearance -> Widgets menu.
1. Drag either the "Gravy Events by Mood" or "Gravy Events by Channel" to one of the theme's widget areas to activate it.
1. For "Gravy Events by Channel": Fill in the title, City, State or Zip, choose a  Channel and Layout style and save. Title is optional.
1. For "Gravy Events by Mood": Fill in the title, City, State or Zip, choose a Mood and Layout style and save. Title is optional.
